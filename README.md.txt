# KantinPay ITK (OOP Kotlin)

## Deskripsi
Proyek simulasi sistem e-wallet kantin berbasis Object-Oriented Programming (OOP) dengan penerapan enkapsulasi.

## Fitur
- Validasi PIN
- Validasi saldo
- Validasi stok
- Enkapsulasi data sensitif

## Struktur Folder
- src/ → kode program
- docs/ → diagram UML
- screenshots/ → bukti hasil program

## Cara Menjalankan
1. Buka di IntelliJ IDEA / Kotlin Playground
2. Jalankan file Main.kt

## Contoh Output
Gagal: Saldo tidak cukup  
Berhasil membeli Ayam Geprek  
Gagal: Stok habis  
Gagal: PIN salah
